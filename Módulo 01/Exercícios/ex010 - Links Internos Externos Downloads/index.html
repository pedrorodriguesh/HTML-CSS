Você pode usar o meu repositório público no GitHub

Para fazer o hyperlink usa a tag <a> que é ANCHOR, na parte de href precisa colocar a URL completa.

para abrir o link externo em uma OUTRA aba use, após a URL, target= "_blank" e na frente rel= "external"
